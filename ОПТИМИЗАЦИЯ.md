**БЫЛО:**

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### MessagePack сериализация

**Почему быстро:**
- Бинарный формат (не JSON)
- Нулевые аллокации при десериализации
- Поддержка объектов MessagePackObject

**Оптимизация:**
```csharp
// Сериализуем 1 раз, отправляем 1 раз
var blockData = MessagePackSerializer.Serialize(currentBlock);
var resultMessage = new NetworkMessage(MessageType.BlockResult, blockData);
await SendToAsync(resultMessage, coordinatorIp, coordinatorPort);
```

---

### UDP коммуникация

**Почему быстро:**
- Без установления соединения (connectionless)
- Минимальные накладные расходы
- Асинхронная отправка/получение

**Оптимизация:**
```csharp
// Отправляем все клетки параллельно
var sendTasks = workers.Select(w => SendToAsync(...)).ToList();
await Task.WhenAll(sendTasks);
```

---

### Клеточное разбиение

**Оптимальный размер клетки:**

| Размер матрицы | Воркеров | Размер клетки |
|----------------|----------|---------------|
| 100×100 | 4 | 50×50 |
| 500×500 | 4 | 250×250 |
| 1000×1000 | 4 | 500×500 |
| 5000×5000 | 4 | 2500×2500 |

**Формула:**
```csharp
int gridSize = Math.Min((int)Math.Ceiling(Math.Sqrt(numWorkers)), 4);
int blockSize = (n + gridSize - 1) / gridSize;
```
