
### 2. Запуск воркеров
1. Нажмите **"Запустить :11001"** → Откроется окно воркера
2. Нажмите **"Запустить :11002"** → Ещё одно окно
3. Проверьте: **"Активные: 2"**

### 3. Генерация системы
1. Размер: `4`
2. Seed: `42`
3. Нажмите **"Сгенерировать"**

1. **"Последовательное решение"** → Решение: [1, 1, 1, 1]
2. **"Распределённое решение"** → С воркерами
3. **"Сравнить методы"** → Ускорение

## ✅ ПРОВЕРКА

**Генерация:**
```bash
curl -X POST http://localhost:5000/api/generate ^
  -H "Content-Type: application/json" ^
  -d "{\"size\":4,\"seed\":42}"
```

**Последовательное решение:**
```bash
curl -X POST http://localhost:5000/api/solve/sequential ^
  -H "Content-Type: application/json" ^
  -d "{\"size\":4,\"matrix\":[[4,1,1,1],[1,4,1,1],[1,1,4,1],[1,1,1,4]],\"vector\":[7,7,7,7]}"
```

**Ответ:**
```json
{
  "success": true,
  "solution": [1, 1, 1, 1],
  "timeMs": 0,
  "residual": 0
}
```
